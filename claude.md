# Seagull Tech Solutions Ltd Website

## Overview
This is the company website for Seagull Tech Solutions Ltd, a platform engineering consultancy specialising in data platforms, SRE, and cloud infrastructure on GCP/AWS.

## Tech Stack
- Static HTML/CSS website
- Tailwind CSS (via CDN)
- Google Fonts (Work Sans, Inter)
- Formspree for contact form handling
- Hosted on GitHub Pages with custom domain: seagulltech.co.uk
- Dark mode support with localStorage persistence

- **Contact**: Formspree-powered contact form with navy blue background

## Design Principles
- Professional consultancy tone (not personal portfolio)
- Focus on services, projects, and client outcomes rather than individual background
- Real metrics and specific technologies to demonstrate capability
- UK English spelling throughout
- Demonstrates capability to deliver complex platform projects
- Clean, minimal design inspired by professional consultancies (BCG style)
- Light font weights for refined, modern aesthetic

## Typography
- **Headings**: Work Sans (medium/semibold weight)
- **Body text**: Inter
- Font weights intentionally kept light for a clean, professional look
- No heavy/black fonts used
